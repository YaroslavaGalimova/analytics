## Теория вероятностей

Директория: `probability-labs/`

### Лабораторная работа 1

**Директория:** `probability-labs/lab-1/`

**Описание:** Реализация различных методов вычисления вероятностей биномиального распределения и их сравнение.
  - Прямое вычисление по биномиальной формуле
  - Аппроксимация Пуассона
  - Локальная теорема Муавра-Лапласа
  - Интегральная теорема Муавра-Лапласа

### Лабораторная работа 2

**Директория:** `probability-labs/lab-2/`

**Описание:** Моделирование случайных величин и анализ их распределений.
Генерация случайных чисел из заданного распределения различными способами

## Математическая статистика

Директория: `statistics-labs/`

### Лабораторная работа 1

**Директория:** `statistics-labs/lab1/`

**Описание:** Исследование выборочных характеристик экспоненциального распределения.
- Выборочные среднее, дисперсия, медиана
- Характеристики выборочных величин
- Визуализация (гистограмма, violin-plot, box-plot)
- Анализ датасета

### Лабораторная работа 2

**Директория:** `statistics-labs/lab2/`

**Описание:** Статистический анализ данных.
- Применение методов статистической обработки данных
- Вычисление выборочных величин (Байесовская оценка, ММП)

### Лабораторная работа 3

**Директория:** `statistics-labs/lab3/`

**Описание:** Доверительные интервалы
- Построение доверительных интервалось для выборочных величин
- Анализ и интерпретация результатов

## Машинное обучение

Директория: `ml/`

### Лабораторная работа 1: Обработка данных

**Файл:** `ml/ml-lab1-process-data.ipynb`

**Описание:** Работа с данными для задач машинного обучения.
- Загрузка и предобработка данных
- Работа с различными форматами данных (JSON, ARFF)
- Exploratory Data Analysis (EDA)
- Подготовка данных для моделирования

**Используемые библиотеки:**
- pandas
- numpy
- requests
- lxml
- arff

### Лабораторная работа 2: Линейные методы

**Файл:** `ml/ml-lab2-linear-methods.ipynb`

**Описание:** Реализация и применение линейных методов машинного обучения.

**Основные задачи:**
- Реализация линейных моделей (градиентный спуск с регуляризацией, линейный поиск)
- Проверка на разных функциях риска
- Обучение и валидация моделей

## Домашние задания T-Bank Аналитика

Директория: `t-bank-analytics-homeworks/`

Домашние задания по программе T-Bank (Аналитика) включают задачи по:

### Homework 1: Теория вероятностей
**Файл:** `homework1_probability.ipynb`

**Темы:**
- Работа с различными распределениями (биномиальное, Пуассона, экспоненциальное, нормальное, равномерное, Вейбулла)
- Вычисление вероятностей и статистик
- Визуализация распределений

### Homework 2: Математическая статистика
**Файл:** `homework2_statistics.ipynb`

**Темы:**
- Статистический анализ данных
- Проверка гипотез
- Оценка параметров

### Homework 3
**Файл:** `homework3.ipynb`

**Темы:**
- Проверка гипотез различными методами
- Дизайн эксперимента
- Анализ ошибок 1 и 2 рода

### KNN Homework
**Файл:** `knn_homework.ipynb`

**Описание:** Реализация и применение метода k-ближайших соседей (k-Nearest Neighbors)

**Основные задачи:**
- Реализация алгоритма KNN
- Подбор оптимального значения k
- Оценка качества модели

### Домашнее задание (обработка данных с использованием python)
**Файл:** `domashnee-zadanie.ipynb`
